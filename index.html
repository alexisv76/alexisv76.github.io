Projet immobilier

Application réalisée avec Flask répondant à la demande d’une entreprise d’obtenir les biens immobiliers à rénover en Normandies afin de les proposer a ses clients. Les logements sont obtenus par scrapping grâce aux bibliothèques sélénium et beautifulsoup. Toutes les données sont gérées avec peewee et stockées dans une base de donnée postgrés. Les clients peuvent trouver les biens sur l’applications grâce à des fonctions de recherches précise. Un planificateur permet l’automatisation de l’application en apportant chaque jour de nouvelles données, en sauvegardant les données sous excel avec pandas et en vérifiant la validité des données

Projet arbre de compétence

Application réalisée avec Flask permettant le suivit des competences des élèves de la formation développeur data du Digifab. Le module networkx permet d’obtenir un rendu visuel des compétences via un système de noeud et d’avoir une classification de celles-ci. Les données sont gérées et stockées dans une base de donnée relationnelle grâce à la bibliothèque Sqlite3. Pour effectuer les premiers test nous avons utilisés Numpy et random pour obtenir des données fictives.

Projet borne de recharge

Application réalisée avec Flask créée pour les utilisateurs de véhicule électrique. Elle a pour but de permettre de trouver les bornes de recharges proche de notre emplacement. Postgres est utilisé pour stocker et gérer les données, le back end est en langage python. Ce projet m'a permis d'obtenir mes certification de niveau BAC +2.
